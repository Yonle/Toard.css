Short Documentation

Only some simple classes that you can remember them easily. The rest is you the one who will design your site.

Design

Documentations

  • left, right, top, bottom, absolute, relative, fixed

    It's all about position. You know how it works once you did it.

  • border

    A simple class that create a border.

  • popup

    Just like the class name said. It's a popup class. In other meaning, It's using element:target in newer CSS to work.

popup class usage example

    <a href="#to_popup">
      Click me!
    </a>
      
    <!-- Here's the popup element. It's hidden until it's being clicked. --->
    <div class="border popup" id="to_popup">
      <h3>Hello world!</h3>
    </div>
